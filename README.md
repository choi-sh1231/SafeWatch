# 🛡️ SafeWatch: 주간 산업재해 발생 위험 예측 및 조기경보 플랫폼

SafeWatch는 고용노동부 산업재해 통계, 국민권익위원회 민원 빅데이터 분석 API, 고용노동부 사업장 현황 데이터를 결합해 시도×산업×주차 단위의 주간 산재 발생 위험을 예측하고, 민원 신호를 보조적으로 반영해 정상·주의·위험 3단계 경보와 우선 점검 대상 Top5를 제공하는 웹사이트형 플랫폼입니다.

## 🎯 프로젝트 목적

- 산재 승인일 기준 공식 통계의 시간 지연 한계를 민원 데이터로 보완
- 안전 감독관과 기업 안전관리자가 한정된 점검 자원을 위험 지역·산업에 우선 배분하도록 지원
- 사후 통계 중심 정책에서 벗어나 **사전 예측형 산재 조기경보 서비스** 구현

## 📚 데이터 및 분석 방법론

### 활용 데이터

| 데이터 | 출처 | 활용 목적 |
|--------|------|-----------|
| 산재보험 최초 요양 승인 정보 | 근로복지공단 (국가중점데이터) | 시도×산업×주차 단위 산재 발생 건수 산출 |

| 사업장 현황 | 고용행정통계 | 시도×산업별 사업장 규모 산출 및 노출량 보정 |

### 담당 역할 — 통계 검정 및 모델링

**통계 검정**
- Granger 인과성 검정 + FDR 다중검정 보정으로 민원 신호의 산재 선행성을 **11개 시도에서 통계적으로 검증**
- leading/post 키워드 분리로 인과 방향 오류 방지
- Shapiro 정규성 진단 후 Welch t / Mann-Whitney U 적용
- 산재 0건 주차까지 명시한 영구 패널 구조 구축으로 모델 과대평가 방지

**모델링**
- 통합 모델 대신 **시도별 개별 XGBoost 모델** 설계 (11개 시도)
- 나무 500개, 학습률 0.03, 최대 깊이 5, 행/열 샘플링 0.85
- NSR(Noise to Signal Ratio) 기반 임계값 최적화 (FPR/TPR 비율 그리드 탐색)
- 예측값을 시도별 70분위·90분위 기준 **정상·주의·위험 3단계 경보**로 변환

### 데이터 전처리
- 재해일자 → 월요일 시작 주차 변환
- 사업장 소재지 → 17개 시도 표준화
- 업종명 → 18개 거시 산업 매핑
- 민원 데이터 → 주간 패널 변환, leading/post 키워드 분리

## 📊 주요 결과

- **11개 시도**에서 민원 신호의 산재 선행성 통계 검증 완료
- 위험 산업 31개·주의 산업 69개 주간 자동 산출
- 우선 점검 Top5 자동 도출 (시도×산업 조합 기준)
- 시도별 모델 구조로 통합 모델 대비 RMSE·NSR 개선
- SHAP Tree Explainer 기반 설명 가능성 설계 (고도화 예정)
